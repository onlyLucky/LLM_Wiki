> **适用场景**：AI生成网页PPT式视频 + 多平台图文分发

> **版本**：v1.0 | 2026-06-10


## 二、第一阶段：0→1 冷启动期

### 2.1 选题策划

#### 目标
确定有流量潜力且符合账号定位的内容方向。

#### 执行步骤

| 步骤 | 动作 | 工具/方法 |
|------|------|----------|
| 1 | 定位目标受众画像 | 年龄、性别、兴趣、痛点 |
| 2 | 挖掘爆款选题 | 5118、新榜、蝉妈妈、各平台热搜榜 |
| 3 | 竞品分析 | 拆解同领域Top10账号的爆款内容 |
| 4 | 建立选题库 | 飞书/Notion表格，储备30+选题 |
| 5 | 选题评分筛选 | 按「需求度×竞争度×变现力」打分 |

#### 选题公式
```
爆款选题 = 目标人群痛点 + 高情绪价值 + 信息增量 + 时效性
```

#### 各平台选题偏好

| 平台 | 选题特点 | 内容形式 |
|------|---------|---------|
| 公众号 | 深度长文、观点输出、行业洞察 | 3000-5000字图文 |
| 小红书 | 实用干货、种草测评、生活方式 | 图文笔记/短视频 |
| B站 | 知识科普、深度评测、系列内容 | 中长视频(5-20min) |
| 抖音 | 强钩子开头、情绪共鸣、信息密度 | 短视频(15-60s) |
| YouTube | SEO导向、长尾内容、系列播放列表 | 中长视频(8-15min) |
| TikTok | 趋势跟随、强视觉冲击、快节奏 | 短视频(15-60s) |

### 2.2 大纲设计

#### 目标
构建内容骨架，确保逻辑清晰、信息完整。

#### 标准大纲模板

```markdown
# 标题：[主标题] + [副标题/钩子]

## 一、开篇钩子（前3秒/前30字）
- 痛点提问 / 数据冲击 / 反常识观点 / 故事引入

## 二、核心内容（主体）
### 2.1 背景/问题定义
### 2.2 解决方案/核心观点（分3-5个点）
   - 要点1：...
   - 要点2：...
   - 要点3：...
### 2.3 案例/数据支撑

## 三、结尾升华
- 金句总结 / 行动号召(CTA) / 互动引导

## 四、附加元素
- 封面图概念
- 关键数据可视化
- 互动问题
```

#### 各平台大纲差异

| 平台 | 大纲特点 |
|------|---------|
| 公众号 | 深度结构，允许层层递进，可设悬念 |
| 小红书 | 前置结论，「总-分-总」，多用emoji分段 |
| B站 | 脚本化结构，需标注节奏点、梗点、转场 |
| 抖音 | 黄金3秒法则，每5秒一个信息点，快节奏 |
| YouTube | SEO标题+章节标记(Timestamps)，信息密度适中 |
| TikTok | 强视觉冲击大纲，每2-3秒一个画面切换 |

### 2.3 文案撰写

#### 目标
产出高转化、高完播的文案内容。

#### 文案结构公式

**图文平台（公众号/小红书）**：
```
标题：数字法/痛点法/悬念法/对比法
开头：SCQA模型（情境-冲突-问题-答案）
正文：金字塔原理，结论先行
结尾：CTA + 互动引导
```

**视频平台（B站/抖音/YouTube/TikTok）**：
```
0-3秒：钩子（反常识/痛点/好奇/恐惧）
3-15秒：问题定义/背景铺垫
15-60秒：核心内容（3-5个要点）
结尾：CTA（关注/点赞/评论/转化）
```

#### 文案撰写SOP

| 步骤 | 动作 | 输出物 |
|------|------|--------|
| 1 | 根据大纲扩写初稿 | 完整文案文档 |
| 2 | 口语化改造（视频用） | 口播稿 |
| 3 | 植入钩子与节奏点 | 标注版文案 |
| 4 | 添加平台适配元素 | 多平台版本 |
| 5 | 内部审核/AI优化 | 终稿 |

#### 文案优化检查清单

- [ ] 前3秒/前30字是否有强钩子？
- [ ] 是否有明确的受众指向？
- [ ] 信息密度是否适中（不过载/不空洞）？
- [ ] 是否有情绪起伏（好奇→共鸣→收获）？
- [ ] CTA是否清晰且低门槛？
- [ ] 是否植入关键词（SEO）？

### 2.4 脚本分镜

#### 目标
将文案转化为可执行的视频制作蓝图。

#### 分镜脚本模板

```markdown
# 视频分镜脚本

## 基础信息
- 视频标题：
- 目标时长：
- 目标平台：
- 风格调性：

## 分镜表

| 镜号 | 时间 | 画面内容 | 旁白/口播 | 字幕 | BGM/音效 | 备注 |
|------|------|---------|----------|------|---------|------|
| 1 | 0-3s | 封面页/标题动画 | 钩子台词 | 主标题 | 节奏感强 | 抓住注意力 |
| 2 | 3-8s | 问题场景（PPT页1） | 痛点描述 | 关键词高亮 | 持续 | 建立共鸣 |
| 3 | 8-20s | 核心观点（PPT页2-3） | 要点1+2 | 逐字出现 | 轻音乐 | 信息传递 |
| ... | ... | ... | ... | ... | ... | ... |
| n | 结尾 | CTA页面 | 引导关注 | 二维码/ID | 品牌音效 | 转化 |
```

#### AI生成PPT式视频的分镜要点

由于画面由AI生成网页PPT方式呈现，分镜需特别注意：

1. **页面数量控制**：1分钟视频约8-12页PPT
2. **视觉节奏**：每页停留3-5秒，关键页可延长至8秒
3. **动画规划**：标注每页的进入/退出动画类型
4. **转场设计**：页面间切换方式（淡入/滑动/缩放）
5. **统一风格**：配色、字体、布局的一致性

### 2.5 画面生成（AI网页PPT）

#### 目标
通过AI工具高效生成统一风格的视频画面。

#### 推荐工具链

| 环节 | 工具选项 | 特点 |
|------|---------|------|
| PPT生成 | Gamma.app / 美图AI PPT / 讯飞智文 | AI一键生成，支持主题定制 |
| 视觉设计 | Canva / 稿定设计 | 模板丰富，适合快速出图 |
| AI绘图 | Midjourney / 即梦 / 可灵 | 生成定制化配图 |
| 动画制作 | 剪映 / 必剪 / CapCut | 一键成片，多平台适配 |

#### 画面生成SOP

```
Step 1: 确定视觉风格
   ↓ 配色方案 + 字体规范 + 布局模板
Step 2: 按分镜制作单页
   ↓ 每页包含：标题 + 核心信息 + 视觉元素
Step 3: 统一视觉规范
   ↓ 检查配色/字体/间距一致性
Step 4: 导出为图片序列
   ↓ PNG/JPG，16:9或9:16
Step 5: 导入剪辑软件
   ↓ 添加动画、转场、配音
```

#### 画面设计原则

1. **一页一重点**：每页只传达一个核心信息
2. **大字报思维**：标题字号足够大，3米外能看清
3. **留白美学**：信息密度适中，不要填满
4. **视觉层次**：标题 > 核心数据 > 辅助说明
5. **品牌一致性**：Logo、配色、字体统一

### 2.6 话本稿（配音/口播稿）

#### 目标
产出自然流畅、有感染力的配音脚本。

#### 话本稿与文案的区别

| 维度 | 文案稿 | 话本稿 |
|------|--------|--------|
| 语言风格 | 书面语、可阅读 | 口语化、可听懂 |
| 句式结构 | 长句、复合句 | 短句、简单句 |
| 信息密度 | 高，可回看 | 适中，一遍懂 |
| 节奏标记 | 无 | 停顿、重音、语速 |
| 互动设计 | 弱 | 强（提问、留白） |

#### 话本稿模板

```markdown
# 话本稿：[视频标题]

【开场】0:00-0:05
（轻快BGM起）
你有没有遇到过这样的情况——[痛点场景描述]？
（停顿1秒）
今天这条视频，我给你一个完整的解决方案。

【正文】0:05-0:45
首先，我们要明白...[要点1]
（页面切换音效）
具体来说，有三个关键点：
第一，...[要点2] —— 这里要注意...[补充说明]
第二，...[要点3]
第三，...[要点4]

【案例】0:45-0:55
比如...[具体案例，增强可信度]

【结尾】0:55-1:00
所以，记住这个公式：[金句总结]
（BGM渐强）
如果你觉得有用，点赞收藏，我们下期见！
```

| ElevenLabs | 英文超自然、情绪丰富 | YouTube/TikTok |

### 2.7 多平台输出

#### 目标
一稿多用，适配各平台特性分发。

#### 输出矩阵

```
原始内容（大纲+文案+脚本）
    │
    ├──→ 公众号：长图文（3000-5000字）
    │
    ├──→ 小红书：图文笔记（1000字内）+ 短视频（60s内）
    │
    ├──→ B站：中长视频（5-15分钟）+ 专栏文章
    │
    ├──→ 抖音：短视频（15-60秒）+ 图文卡片
    │
    ├──→ YouTube：中长视频（8-15分钟）+ 社区图文
    │
    └──→ TikTok：短视频（15-60秒）+ 系列内容
```

#### 各平台输出规范

| 平台 | 格式 | 尺寸 | 时长 | 特殊要求 |
|------|------|------|------|---------|
| 公众号 | 图文 | 900×383(封面) | 无限制 | 原创声明、合集标签 |
| 小红书 | 图文/视频 | 3:4或9:16 | 图文≤15张，视频≤5min | 关键词布局、话题标签 |
| B站 | 视频 | 16:9 | 1-30min | 分P、弹幕引导、三连 |
| 抖音 | 视频 | 9:16 | 15s-10min | 前3秒钩子、音乐热点 |
| YouTube | 视频 | 16:9 | 8-15min(最佳) | 缩略图、标题SEO、章节 |
| TikTok | 视频 | 9:16 | 15s-3min | 趋势音乐、Hashtag |

#### 内容适配原则

1. **标题适配**：
   - 公众号：深度感，「为什么」「如何」「揭秘」
   - 小红书：实用感，「保姆级」「手把手」「亲测」
   - 抖音：冲突感，「千万别」「震惊」「原来」
   - YouTube：SEO导向，含关键词，数字+年份

2. **封面适配**：
   - 小红书：3:4竖图，大字报风格，信息密度高
   - B站/YouTube：16:9横图，人物+标题，情绪表达
   - 抖音/TikTok：视频首帧即封面，前3秒定生死

3. **文案适配**：
   - 长平台（公众号/B站专栏）：完整论述，引用数据
   - 短平台（小红书/抖音）：结论先行，要点罗列

### 4.2 AB测试体系

#### 测试维度

```markdown
1. 标题测试
   - 变量：钩子类型、数字、情绪词
   - 样本：同内容，不同标题，各推1000曝光

2. 封面测试
   - 变量：配色、人物、文字排版
   - 样本：同视频，不同封面，观察CTR

3. 开头测试
   - 变量：3秒钩子类型
   - 样本：同脚本，不同开场，观察完播率

4. 时长测试
   - 变量：15s / 30s / 60s
   - 样本：同内容，不同剪辑节奏

5. CTA测试
   - 变量：关注引导 / 评论引导 / 转化引导
   - 样本：观察不同CTA的转化效果
```

### 4.4 内容升级路径

```
Level 1：搬运/拼凑 → 无差异化，难持续
    ↓
Level 2：伪原创/改写 → 有基本价值，竞争大
    ↓
Level 3：原创整合 → 有信息增量，开始起量
    ↓
Level 4：独家观点 → 有认知差，建立信任
    ↓
Level 5：IP化内容 → 人格魅力，粉丝忠诚
    ↓
Level 6：品牌内容 → 行业标准，话语权
```

## 七、常见问题FAQ

**Q1：新手应该先专注哪个平台？**
> 建议根据内容形式选择：图文优先→小红书/公众号；视频优先→抖音/B站。选一个主攻，其他分发。

**Q2：AI生成的内容会被平台限流吗？**
> 平台限制的是低质内容，不是AI内容。关键在于：是否有信息增量、是否满足用户需求、是否原创整合。

**Q3：日更还是周更？**
> 冷启动期建议日更（养号+测试），稳定后可改为高质量周更3-5篇。质量>数量，但数量是测试前提。

**Q4：如何判断内容方向是否正确？**
> 看数据反馈：连续10篇内容，如果互动率持续低于行业均值，需要调整方向；如果偶有爆款，说明方向对，需要优化执行。

**Q5：一个人能运营多平台吗？**
> 可以，但需要工具提效。建议：1个主平台深度运营 + 2-3个平台自动化分发。不要贪多。
