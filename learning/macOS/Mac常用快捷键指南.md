## 10. 本机个性化设置

> 以下为当前设备的实际配置，基于 `defaults read` 读取

### 10.1 键盘速度设置

| 配置项 | 当前值 | 说明 |
|:---|:---|:---|
| KeyRepeat | 2 | 按键重复速度（越小越快） |
| InitialKeyRepeat | 15 | 首次重复延迟（越小越快） |
| ApplePressAndHoldEnabled | 0 | 长按弹出字符选择（已禁用） |

**说明：**
- 系统默认 KeyRepeat = 6，当前设置为 2（较快）
- 系统默认 InitialKeyRepeat = 25，当前设置为 15（较短延迟）
- 长按字符选择已禁用，长按会直接重复输入

### 10.2 系统快捷键状态

| 功能 | 状态 | 默认快捷键 |
|:---|:---:|:---|

## 附录：快捷键学习建议

### 优先级学习路径

**第一阶段（立即使用）**
- `⌘ + C/V/X/Z` - 复制/粘贴/剪切/撤销
- `⌘ + Tab` - 应用切换
- `⌘ + Space` - Spotlight 搜索
- `⌘ + W/Q` - 关闭/退出

**第二阶段（提高效率）**
- `⌘ + ⇧ + 3/4/5` - 截图
- `⌘ + ←→` - 行首行尾
- `⌥ + ←→` - 按词移动
- `⌘ + Delete` - 删除文件

**第三阶段（高级操作）**
- `⌃ + A/E/U/K` - 终端操作
- `⌘ + ⌥ + ⎋` - 强制退出
- `⌃ + ↑↓←→` - 空间管理
- `⌘ + ⇧ + .` - 显示隐藏文件

### 自定义快捷键方法

1. 打开 系统偏好设置 > 键盘 > 快捷键
2. 选择左侧分类（如"应用程序快捷键"）
3. 点击 "+" 添加自定义快捷键
4. 输入菜单标题和快捷键组合
