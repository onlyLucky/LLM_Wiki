# Three.js + GLSL + WebGPU 学习计划

> 目标：做出 3D 特效创意网站
> 当前水平：L1（零基础）→ 目标：L4（能做特效创意）
> 每周投入：20 小时，周期：5 周
> 课程次数：20 次 × 2 小时

---

## 分层级：五级路径

| 等级             | 描述                 | 核心能力                                           | 通关标准                         |
| -------------- | ------------------ | ---------------------------------------------- | ---------------------------- |
| **L1** 零基础     | 能画出一个旋转的立方体        | 场景/相机/渲染器/灯光/材质基础                              | 独立搭建场景，不查文档写出基础 demo         |
| **L2** 能做 demo | 能跟着教程做出可交互的 3D 场景  | 模型加载、动画、OrbitControls、基础后处理                    | 能把教程项目改出自己的效果                |
| **L3** 能做项目    | 能独立从零搭建一个完整的 3D 网页 | 自定义 ShaderMaterial、后处理链、性能优化、多场景切换             | 独立做出一个加载流畅、效果完整的单页 3D 展示     |
| **L4** 能做特效创意  | 能做出 award 级别的视觉效果  | 高级 GLSL（噪声、SDF、ray marching）、粒子系统、GSAP + 3D 联动 | 能复刻一个 Awwwards 获奖网站的核心 3D 效果 |
| **L5** 能做技术指导  | 能设计复杂 3D 架构        | WebGL 底层管线、WebGPU 迁移、自定义渲染管线                   | 能带队做一个完整的 3D 产品              |
