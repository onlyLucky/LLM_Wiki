## 核心知识点

### 1. 几何体（Geometry）是什么

几何体定义了物体的**形状**。它由顶点（vertices）和面（faces）组成。

```
几何体 = 顶点坐标 + 面的连接方式
```

Three.js 内置了多种基础几何体，不需要手动定义顶点数据。

### 2. 五种基础几何体

| 几何体 | 用途 | 构造函数 |
|--------|------|----------|
| BoxGeometry | 立方体/长方体 | `new THREE.BoxGeometry(w, h, d)` |
| SphereGeometry | 球体 | `new THREE.SphereGeometry(r, wSeg, hSeg)` |
| CylinderGeometry | 圆柱体/圆锥体 | `new THREE.CylinderGeometry(rTop, rBot, h, seg)` |
| TorusGeometry | 圆环/甜甜圈 | `new THREE.TorusGeometry(r, tube, rSeg, tSeg)` |
| PlaneGeometry | 平面/地面 | `new THREE.PlaneGeometry(w, h, wSeg, hSeg)` |

### 3. 分段数（Segments）— 性能与质量的平衡

分段数决定了几何体的**精细程度**：

| 分段数 | 效果 | 顶点数 | 适用场景 |
|--------|------|--------|----------|
| 低（8） | 粗糙，能看到多边形 | 少 | 低性能设备、像素风格 |
| 中（32） | 圆滑 | 适中 | 大多数场景 |
| 高（64+） | 非常圆滑 | 多 | 特写镜头、高质量渲染 |

**关键原则**：
- 用**最少的分段数**达到**可接受的视觉效果**
- 分段数用 **2 的幂**（8, 16, 32, 64）效果最好
- 远处的物体可以用低分段数，近处用高分段数（LOD 策略）

### 4. Wireframe（线框）可视化

```typescript
const material = new THREE.MeshBasicMaterial({ wireframe: true })
```

线框模式可以直观看到几何体的**顶点和面**，用于：
- 调试几何体形状
- 观察分段数的效果
- 做线框风格的视觉效果

### 5. 组合多个几何体

用 `THREE.Group` 或 `scene.add()` 把多个几何体组合在一起：

```typescript
const group = new THREE.Group()
group.add(mesh1)
group.add(mesh2)
group.add(mesh3)
scene.add(group)

// 移动整个组
group.position.x = 5
group.rotation.y = Math.PI / 4
```

## 参数详解

### BoxGeometry

```typescript
new THREE.BoxGeometry(width, height, depth, widthSegments, heightSegments, depthSegments)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| width | 1 | X 轴宽度 |
| height | 1 | Y 轴高度 |
| depth | 1 | Z 轴深度 |
| widthSegments | 1 | X 轴分段数 |
| heightSegments | 1 | Y 轴分段数 |
| depthSegments | 1 | Z 轴分段数 |

### SphereGeometry

```typescript
new THREE.SphereGeometry(radius, widthSegments, heightSegments)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| radius | 1 | 球体半径 |
| widthSegments | 32 | 水平分段数（经度方向） |
| heightSegments | 16 | 垂直分段数（纬度方向） |

### CylinderGeometry

```typescript
new THREE.CylinderGeometry(radiusTop, radiusBottom, height, radialSegments)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| radiusTop | 1 | 顶部半径 |
| radiusBottom | 1 | 底部半径 |
| height | 1 | 高度 |
| radialSegments | 32 | 径向分段数 |

**特殊用法**：
- `radiusTop = 0` → 圆锥体
- `radiusTop = radiusBottom` → 圆柱体
- `radiusTop ≠ radiusBottom` → 圆台

### TorusGeometry

```typescript
new THREE.TorusGeometry(radius, tube, radialSegments, tubularSegments)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| radius | 1 | 圆环半径（中心到管中心） |
| tube | 0.4 | 管的粗细 |
| radialSegments | 12 | 径向分段数 |
| tubularSegments | 48 | 管的分段数 |

### PlaneGeometry

```typescript
new THREE.PlaneGeometry(width, height, widthSegments, heightSegments)
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| width | 1 | 宽度 |
| height | 1 | 高度 |
| widthSegments | 1 | 宽度分段数 |
| heightSegments | 1 | 高度分段数 |

**注意**：Plane 默认在 XY 平面，要做地面需要旋转：
```typescript
plane.rotation.x = -Math.PI / 2  // 旋转到水平
```

---

## 常见错误

1. **分段数太高**：球体用 128×128 分段，性能浪费。通常 32×16 足够。
2. **分段数太低**：球体用 4×4 分段，看起来像多面体，不圆滑。
3. **忘记旋转 Plane**：Plane 默认垂直，做地面需要 `rotation.x = -Math.PI/2`。
4. **不理解 segments 参数顺序**：SphereGeometry 的 widthSegments 是水平方向，heightSegments 是垂直方向。
5. **几何体和材质混淆**：Geometry 只定义形状，Material 定义外观，两者分开。

## 代码示例

### 创建一个球体

```typescript
const geometry = new THREE.SphereGeometry(1, 32, 16)
const material = new THREE.MeshStandardMaterial({ color: 0x4488ff })
const sphere = new THREE.Mesh(geometry, material)
scene.add(sphere)
```

### 创建一个低分段线框球体

```typescript
const geometry = new THREE.SphereGeometry(1, 8, 4)
const material = new THREE.MeshBasicMaterial({ wireframe: true, color: 0xffd43b })
const wireSphere = new THREE.Mesh(geometry, material)
scene.add(wireSphere)
```

### 组合多个几何体

```typescript
const group = new THREE.Group()

const box = new THREE.Mesh(new THREE.BoxGeometry(1, 1, 1), material)
const sphere = new THREE.Mesh(new THREE.SphereGeometry(0.5, 32, 16), material)

group.add(box)
group.add(sphere)
sphere.position.x = 2  // 相对于 group 的位置

scene.add(group)
group.position.set(0, 1, 0)  // 移动整个组
```
