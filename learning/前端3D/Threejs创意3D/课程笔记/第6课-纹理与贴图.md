## 🎯 核心概念

### 1. 纹理类型对比

| 纹理类型 | 作用 | 加载方式 | 适用场景 |
|----------|------|----------|----------|
| 图片纹理 | 给几何体表面贴图 | TextureLoader | 皮肤、材质、贴花 |
| 立方体纹理 | 6 面立方体包裹 | CubeTextureLoader | 天空盒、环境反射 |
| HDRI 纹理 | 高动态范围环境图 | RGBELoader | 环境光照、反射 |
| Canvas 纹理 | 程序生成纹理 | CanvasTexture | 动态纹理、数据可视化 |

### 2. UV 坐标系统

**定义**：决定纹理如何映射到几何体表面

```
V (1,1) ┌─────────┐
        │         │
        │  纹理   │
        │         │
   (0,0)└─────────┘ U (1,0)
```

- U：水平方向（0 → 1）
- V：垂直方向（0 → 1）
- (0,0)：左下角
- (1,1)：右上角

### 3. 纹理变换参数

| 参数 | 作用 | 示例 |
|------|------|------|
| repeat | 纹理重复次数 | `texture.repeat.set(2, 2)` |
| offset | 纹理偏移量 | `texture.offset.set(0.5, 0)` |
| rotation | 纹理旋转 | `texture.rotation = Math.PI / 4` |
| wrapS/wrapT | 环绕模式 | `THREE.RepeatWrapping` |

### 4. 环绕模式

| 模式 | 效果 | 适用场景 |
|------|------|----------|
| RepeatWrapping | 重复 | 地面、墙壁 |
| ClampToEdgeWrapping | 边缘拉伸 | 单张贴图 |
| MirroredRepeatWrapping | 镜像重复 | 无缝纹理 |

### 5. CubeTextureLoader 天空盒

**6 面立方体纹理**：
```
px.jpg — 右（+X）
nx.jpg — 左（-X）
py.jpg — 上（+Y）
ny.jpg — 下（-Y）
pz.jpg — 前（+Z）
nz.jpg — 后（-Z）
```

**应用**：
```typescript
const envMap = cubeTextureLoader.load([...])
scene.background = envMap    // 场景背景
scene.environment = envMap   // 环境光照 + 反射
```

### 6. HDRI 环境贴图

**HDRI vs CubeTextureLoader**：
- HDRI：单张图片，球形映射，光照信息更丰富
- CubeTextureLoader：6 张图片，立方体映射，性能更好

```typescript
const rgbeLoader = new RGBELoader()
rgbeLoader.load('/textures/venice-sunset.hdr', (texture) => {
  texture.mapping = THREE.EquirectangularReflectionMapping
  scene.environment = texture
})
```

### 7. 环境贴图反射

**金属度与粗糙度对反射的影响**：
- metalness=1, roughness=0 → 完美镜面反射
- metalness=1, roughness=0.5 → 模糊反射
- metalness=0 → 不反射环境

## 💻 代码实现要点

### 1. TextureLoader 异步加载

```typescript
const textureLoader = new THREE.TextureLoader()
textureLoader.load(
  '/textures/earth-day.jpg',
  (texture) => {
    material.map = texture
    material.needsUpdate = true
  },
  undefined,
  (error) => console.warn('加载失败:', error)
)
```

### 2. Canvas 纹理生成

```typescript
const canvas = document.createElement('canvas')
canvas.width = 512
canvas.height = 512
const ctx = canvas.getContext('2d')!
// 绘制内容...
const texture = new THREE.CanvasTexture(canvas)
```

### 3. 纹理参数控制

```typescript
texture.repeat.set(2, 2)           // 重复
texture.offset.set(0.5, 0)         // 偏移
texture.rotation = Math.PI / 4     // 旋转
texture.wrapS = THREE.RepeatWrapping  // 环绕
```
