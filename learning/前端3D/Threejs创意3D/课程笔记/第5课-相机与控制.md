## 🎯 核心概念

### 1. 透视相机 vs 正交相机

| 特性 | 透视相机 | 正交相机 |
|------|----------|----------|
| 透视效果 | 近大远小 | 远近一样大 |
| 光线汇聚 | 从一个点发散 | 平行（不汇聚） |
| 视锥体形状 | 锥台（近小远大） | 长方体 |
| 适用场景 | 3D 游戏、真实感场景 | CAD、2D 游戏、UI |

**比喻**：
- 透视相机 = 肉眼看世界
- 正交相机 = 工程图纸

### 2. 相机参数

| 参数 | 含义 | 透视相机 | 正交相机 |
|------|------|----------|----------|
| FOV | 视场角（视野角度） | 30°-120° | 无 |
| aspect | 宽高比 | 宽/高 | 宽/高 |
| near | 近裁剪面 | 比这更近不显示 | 比这更近不显示 |
| far | 远裁剪面 | 比这更远不显示 | 比这更远不显示 |
| frustumSize | 可视范围高度 | 无 | 控制可视范围 |

### 3. 视锥体（Frustum）

**定义**：相机能看到的空间范围

**透视相机视锥体**：
- 形状：锥台（金字塔被切掉顶部）
- 特点：近小远大，光线从一个点发散
- 结果：物体「近大远小」

**正交相机视锥体**：
- 形状：长方体
- 特点：远近一样大，光线平行
- 结果：物体远近大小一致

**比喻**：手围成的三角形视野范围

### 4. Near/Far 裁剪面

**问题**：Near/Far 比值过大会导致深度精度下降

**解决方案**：
1. 增大 Far → 覆盖远处物体，但比值过大会导致精度下降
2. 增大 Near → 减小比值，改善精度，但近处物体可能被裁掉
3. 对数深度缓冲 → `renderer.logarithmicDepthBuffer = true`

**对数深度缓冲**：
- 近处精度高，远处精度低
- 完美解决太空场景等超大范围问题

### 5. OrbitControls 轨道控制器

**核心概念**：
- 「轨道」= 相机围绕目标点旋转的路径
- 相机围绕 `controls.target` 旋转
- `enableDamping` = 启用惯性，让旋转更平滑自然

**常用属性**：
```typescript
const controls = new OrbitControls(camera, canvas)
controls.enableDamping = true    // 启用阻尼（惯性）
controls.dampingFactor = 0.05   // 阻尼系数（0-1）
controls.target.set(0, 0, 0)    // 焦点位置
controls.enableZoom = true      // 允许缩放
controls.enableRotate = true    // 允许旋转
controls.enablePan = true       // 允许平移
```

### 6. CameraHelper

**作用**：可视化相机的视锥体

**透视相机 Helper**：锥台形状（近小远大）
**正交相机 Helper**：长方体形状

**用途**：
- 调试相机位置和范围
- 让用户知道相机能看到多大范围

## 💻 代码实现要点

### 1. 相机切换逻辑

**关键问题**：`manager.setCamera()` 会修改 `manager.camera` 的引用

**解决方案**：保存透视相机引用

```typescript
// 保存透视相机引用
const perspectiveCamera = manager.camera as THREE.PerspectiveCamera

// 切换时使用保存的引用
function switchCamera(type: string) {
  if (type === 'perspective') {
    manager.setCamera(perspectiveCamera)  // 使用保存的引用
    cameraHelper.camera = perspectiveCamera
    controls.object = perspectiveCamera
  } else {
    manager.setCamera(orthographicCamera)
    cameraHelper.camera = orthographicCamera
    controls.object = orthographicCamera
  }
}
```

### 2. 参数重置策略

**设计原则**：切换相机时重置所有参数为默认值

```typescript
function switchCamera(type: string) {
  // 重置透视相机
  perspectiveCamera.fov = 50
  perspectiveCamera.near = 0.1
  perspectiveCamera.far = 100
  perspectiveCamera.updateProjectionMatrix()
  
  // 重置正交相机
  orthographicCamera.near = 0.1
  orthographicCamera.far = 100
  updateOrthographicFrustum(10)
  
  // 同步控制面板
  syncSliders()
}
```

### 3. 正交相机视锥体更新

```typescript
function updateOrthographicFrustum(frustumSize: number) {
  const aspect = window.innerWidth / window.innerHeight
  orthographicCamera.left = -frustumSize * aspect / 2
  orthographicCamera.right = frustumSize * aspect / 2
  orthographicCamera.top = frustumSize / 2
  orthographicCamera.bottom = -frustumSize / 2
  orthographicCamera.updateProjectionMatrix()
}
```

### 4. 窗口自适应

```typescript
window.addEventListener('resize', () => {
  const aspect = window.innerWidth / window.innerHeight
  
  // 透视相机：更新 aspect
  perspectiveCamera.aspect = aspect
  perspectiveCamera.updateProjectionMatrix()
  
  // 正交相机：更新 left/right
  updateOrthographicFrustum(currentFrustumSize)
  
  cameraHelper.update()
})
```
