## 🎯 核心概念

### 1. 场景图的树形结构

**定义**：Three.js 中所有物体组织成一棵树，scene 是根节点

```
scene (根节点)
├── sun (太阳)
│   ├── earth (地球)
│   │   └── moon (月球)
│   └── helper (辅助线)
└── lights (灯光)
    ├── ambientLight
    └── directionalLight
```

**关键**：子物体的变换（位置/旋转/缩放）相对于父物体

### 2. 父子关系与变换继承

**核心规则**：
- 子物体的位置是相对于父物体的
- 父物体旋转时，子物体跟着旋转
- 子物体的世界坐标 = 父物体的世界变换 × 子物体的局部变换

**示例**：
```typescript
// 地球距离太阳 5 个单位
earth.position.x = 5

// 月球距离地球 1.5 个单位（不是距离太阳 6.5）
moon.position.x = 1.5

// 构建父子关系
sun.add(earth)   // 地球是太阳的子物体
earth.add(moon)  // 月球是地球的子物体

// 当太阳旋转时，地球和月球一起绕太阳转
// 当地球旋转时，月球绕地球转
```

### 3. 局部坐标系 vs 世界坐标系

| 坐标系 | 说明 | 用途 |
|--------|------|------|
| 局部坐标系 | 物体自身的坐标系 | 设置物体位置、旋转 |
| 世界坐标系 | 场景的全局坐标系 | 获取物体在场景中的实际位置 |

**转换**：
```typescript
// 获取物体的世界坐标
const worldPos = new THREE.Vector3()
object.getWorldPosition(worldPos)

// 获取物体的世界四元数（旋转）
const worldQuat = new THREE.Quaternion()
object.getWorldQuaternion(worldQuat)
```

### 4. Object3D 分组管理

**Group 的作用**：把多个物体组合成一个整体

```typescript
const group = new THREE.Group()
group.add(mesh1)
group.add(mesh2)
group.add(mesh3)
scene.add(group)

// 整体移动
group.position.set(1, 2, 3)

// 整体旋转
group.rotation.y = Math.PI / 4

// 整体缩放
group.scale.set(2, 2, 2)
```

### 5. scene.traverse() 遍历

**作用**：递归遍历场景中的所有物体

```typescript
scene.traverse((child) => {
  if (child instanceof THREE.Mesh) {
    // 对每个 Mesh 执行操作
    child.castShadow = true
    child.receiveShadow = true
  }
})
```

**注意**：traverse 是深度优先遍历

### 6. AxesHelper / GridHelper 调试

**AxesHelper**：显示坐标轴
```typescript
const axesHelper = new THREE.AxesHelper(5) // 长度 5
scene.add(axesHelper)
// 红色 = X 轴，绿色 = Y 轴，蓝色 = Z 轴
```

**GridHelper**：显示网格
```typescript
const gridHelper = new THREE.GridHelper(20, 20) // 大小 20，分割 20
scene.add(gridHelper)
```

## 💻 代码实现要点

### 1. 太阳系父子结构

```typescript
// 创建几何体
const sun = new THREE.Mesh(sunGeo, sunMat)
const earth = new THREE.Mesh(earthGeo, earthMat)
const moon = new THREE.Mesh(moonGeo, moonMat)

// 设置相对位置
earth.position.x = 5    // 距离太阳 5
moon.position.x = 1.5   // 距离地球 1.5

// 构建父子关系
sun.add(earth)
earth.add(moon)
scene.add(sun)
```

### 2. 公转动画

```typescript
// 太阳自转 → 地球绕太阳公转
sun.rotation.y += earthSpeed * delta

// 地球自转 → 月球绕地球公转
earth.rotation.y += moonSpeed * delta
```
