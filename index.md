## 📂 目录结构

```
wiki/
├── SCHEMA.md        ← 规范定义（命名、标签、格式、阈值）
├── index.md         ← 本文件，总目录
├── log.md           ← 操作日志（追加记录，超 500 条轮转）
├── daily/           ← 📅 每日对话汇总（自动生成）
├── entities/        ← 🏢 实体：人物、公司、产品
├── concepts/        ← 💡 概念：技术知识、理论、方法论
├── comparisons/     ← ⚖️ 对比：方案评估、工具选型
├── queries/         ← 🔍 查询：问题追踪、待解事项
├── raw/             ← 📄 原始资料：文章、文档、摘录
├── reading/         ← 📚 读书笔记
├── learning/        ← 📖 学习笔记
├── content/         ← 📝 内容创作中心
│   ├── README.md           ← 内容创作指南
│   ├── topics/             ← 选题库
│   ├── references/         ← 参考资料
│   ├── drafts/             ← 草稿区
│   ├── xiaohongshu/        ← 小红书内容
│   ├── wechat/             ← 公众号内容
│   ├── bilibili/           ← B站内容
│   └── templates/          ← 内容模板
├── assets/          ← 📦 素材库 笔记中的资源都放置在这里
├── templates/       ← 📝 模板
└── _archive/        ← 🗄️ 归档：已过时的页面
```
